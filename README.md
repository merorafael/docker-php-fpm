Docker: PHP, extensions and PHP-FPM
===================================

Official [PHP docker image](https://hub.docker.com/_/php/) with additional extensions.

Supported tags
--------

| PHP Version | Tags            |
| ----------- | --------------- |
| 7.1.1       | `7.1`, `latest` |
| 7.0.15      | `7.0`           |
| 5.6.30      | `5.6`           |
